Repo Reaper

Sign in with a token, pick the repos you want gone, confirm once, they're gone. Deletion is permanent — GitHub has no recycle bin.

01 — Authenticate
Needs a token with the delete_repo scope. Create one at github.com/settings/tokens (classic token — check "delete_repo" and "repo"). The token is kept in memory only, in this browser tab, and is sent directly to api.github.com. Nowhere else.
